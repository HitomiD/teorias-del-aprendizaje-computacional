# teorias-del-aprendizaje-computacional
Archivo de código para la cursada de Teorías del Aprendizaje Computacional - UNSAdA

El propósito del presente repositorio es archivar todo el código y los proyectos realizados durante
la cursada de Teorías del Aprendizaje Computacional en la Universidad Nacional de San Antonio de Areco
para la carrera de Licenciatura en Informática.
